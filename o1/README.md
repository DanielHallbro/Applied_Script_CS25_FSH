# Övning 1

Detta är en del av inlämningen för kursen Applied Script (CS25). Scriptet är en grundläggande övning i Bash-programmering.

## Beskrivning
Scriptet [beskriv kort vad det gör, t.ex. hämtar information om operativsystemet, användaren och aktuell tid].

## Funktioner
* **System-info:** Hämtar data om [t.ex. kernel, hostname].
* **User-context:** Visar vilken användare som kör scriptet.
* **Nätverk:** Visar närverksinformation.
* **Active directory:** Visar vilket directory du befinner dig i.
* **Bakgrundsprocesser:** Visar bakgrundsprocesser
* **Hitta filen:** Skapar en fil, förklarar vart den är, byter directory och ber dig hitta filen.
